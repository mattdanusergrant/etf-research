# etf-research

A simple, mobile-first guide to six robot &amp; AI ETFs, written in plain English
for a general reader — large fonts, high contrast, and no jargon.

The whole site is a single self-contained file: [`index.html`](./index.html).
All CSS and content are inlined — there are no images, no external resources,
and no relative links, so it renders correctly at any URL.

## What's inside

The guide covers six funds, grouped into three colour-coded families:

- **Robot-body funds** — BOTT, HUMN, KOID
- **All-robot funds** — BOTZ, ROBO
- **AI "brain" funds** — ARTY

Each fund gets one card with a plain one-liner, simple fact rows, a
"why people like it" note and a "the risk" note. There's also an at-a-glance
compare section, big-picture takeaways, and a disclaimer.

Numbers are taken from a research note dated May 2026 and will change over time.

## Viewing it

Open `index.html` in any web browser, or visit the published site:

https://mattdanusergrant.github.io/etf-research/
